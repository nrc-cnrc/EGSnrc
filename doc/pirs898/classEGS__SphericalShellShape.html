Detailed Description

A spherical shell shape.

Samples random points within a spherical shell.

A sample input block is given below

:start shape:
    library = egs_spherical_shell
    midpoint = 0, 0, 0
    inner radius = 0.5
    outer radius = 1
    hemisphere = 1  # optional
    half angle = 35 # optional
:stop shape:

If hemisphere is 1 or -1 the shell will be truncated at the z = 0 plane with points being sampled with positive z if hemisphere is 1 and negative z if hemisphere is -1;

If half angle is specified (in degrees) the random points will be sampled within a spherical shell truncated by a conical section with the half angle specified. If half angle is negative the points will sampled with negative z coordinates.
