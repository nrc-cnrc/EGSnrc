Detailed Description

A Fano source.

An Fano source is a source that delivers particles proportional to the mass in the current source region with directions uniformly distributed in emitted from any shape. It is defined using the following input 

:start source:
    library = egs_fano_source
    name = some_name
    :start shape:
        definition of the shape
    :stop shape:
    :start spectrum:
        definition of the spectrum
    :stop spectrum:
    charge = -1 or 0 or 1 for electrons or photons or positrons
    max mass density = 1.2
    geometry = some_name
:stop source:
