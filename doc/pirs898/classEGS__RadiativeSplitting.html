Detailed Description

A radiative splitting object: header.

This ausgab object can be used to increase the efficiency of radiative events. This ausgab object is specified via: 

:start ausgab object:
    library   = egs_radiative_splitting
    name      = some_name
    splitting = n_split
:stop ausgab object:
