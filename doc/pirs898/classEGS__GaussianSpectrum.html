Detailed Description

A Gaussian spectrum.

A Gaussian spectrum is defined in the input file via 

:start spectrum:
    type = Gaussian
    mean energy = the mean kinetic energy
    sigma = the sigma of the spectrum
    or
    fwhm = the full-width-at-half-maximum of the spectrum
:stop spectrum:
