Detailed Description

A shape collection.

This is a shape made from other shapes of any type where the random position is picked from the 's with probabilities defined by the user. A shape collection is defined using 

:start shape:
    library = egs_shape_collection
    :start shape:
        definition of the first shape in the collection
    :stop shape:
    :start shape:
        definition of the second shape in the collection
    :stop shape:
    ...
    :start shape:
        definition of the last shape in the collection
    :stop shape:
    probabilities = p1 p2 ... pn
:stop shape:

A shape collection can be used to pick surface points if all of the shapes in the collection support the getPointSourceDirection() method.
