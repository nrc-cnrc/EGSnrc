Detailed Description

a subclass of EGS_NDGeometry for conveniently defining an RZ geometry

Input blocks for the RZ geometry look like the following:

:start geometry:
    name = phantom
    library = egs_rz
    radii =  1, 2, 3
    z-planes = -4 -3 -2 -1 0 1 2 3 4
    :start media input:
        media = M0
    :stop media input:
:stop geometry:

which would create a num_regions = n_r_bounds*(n_z_bounds - 1) = 3*(9-1) = 24 region geometry with index numbers ranging from 0 to 23.

Alternatively you can specify the geometry using slabs & shells:

:start geometry:
    name = phantom
    library = egs_rz

    number of shells = 2 2 2
    shell thickness =  0.5 1 2

    first plane = -4
    number of slabs = 2 8 2
    slab thickness  = 1 0.5 1

    :start media input:
        media = M0
    :stop media input:
:stop geometry:

Example geometries are available in the rz_lib1.geom and rz_lib2.geom files.
