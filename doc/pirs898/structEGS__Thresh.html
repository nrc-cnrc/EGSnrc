Detailed Description

A structure corresponding to the particle production threshold energies common block THRESH.

Note that the maximum number of media MXMED is defined in the file array_sizes.h that must be present in the user code area. The THRESH common block also contains rmt2 (two times electron rest energy in MeV) and rmsq (electron rest energy squared), that actually have nothing to do with thresholds. Unfortunately, it is too much effort to remove them from THRESH and put them where they belong (rmt2 and rmsq are used in many places).
